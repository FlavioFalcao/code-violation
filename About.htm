Code Violation template,Version 3.0

An application optimized for the iPad that allows code enforcement officials to collect new violations and conduct inspections on existing violations.

You can configure this application with your authoritative data and publish this application on the Apple App Store.

Code Violation application is powered by ArcGIS Server v.10.1.1 update 2 and the iOS SDK.

Mapping API provided by Esri ArcGIS API for iOS
